Como jogar com o Player 2 (p2)

Para se movimentar para frente clique na tecla "w", para se movimentar para trás clique na tecla "s".

Para virar para a direta clique na tecla "d"

Para virar para a esquerda clique na tecla "a"

Para atirar clique na tecla "e"

Obs: Se você bater na borda você ricocheteia
